# minisat
MiniSat is, for now, an integration project of multi prouf of concept block.

### MiniSat Basic Functions.
 - Beacon
 - repeater
 - sensors

### MiniSat Basic Specs.

- microcontroler with BlueTooth 5.2 (STM32WB55)
- power source - battery
